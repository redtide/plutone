@paul wrote:

  <p>
Although Ardour 5.6 contained some really great new features and important fixes, it turned out to contain a number of important regressions compared to 5.5. Some were easily noticed and some were more obscure. Nobody is happy when this happens, and we apologize for any frustration or issues that arose from inadequate testing of 5.6.
  </p>
  <p>
To address these problems, we are making a quick "hotfix" release of Ardour 5.8, which also brings the usual collection of small features and other bug fixes.
  </p>
  <p>
Linux distributions are asked to immediately and promptly replace 5.6 with 5.8 to reduce issues for Ardour users who get the program directly via their software management tools.
  </p>

  <p style="margin:22px 0 22px 0;">
<a class="btn btn-primary btn-large" href="https://community.ardour.org/download">Download&nbsp;&nbsp;<i class="icon-download-alt icon-white"></i></a>
  </p>

  <h2>General Changes and Improvements</h2>
  <ul>
<li>Add Record with Count-in functionality accessible via Transport menu.</li>
<li>Save and restore Location clock modes in Locations pane in Editor List and Locations Window separately.</li>
<li>Improvements to sensitivity of actions in Region context menu. [#7077]</li>
<li>Version keyboard shortcut configuration file (.keys).</li>
<li>Add support for importing MIDI from Pro Tools Session files.</li>
<li>Consistently use the term VCA rather than Control Master.</li>
<li>When starting Ardour without any existing configuration files and not copying them from a previous version then still present the Startup/First time dialog.</li>
<li>Control (Primary Modifier) and right click on group tab will present Group Edit Window.</li>
<li>Shift (Tertiary Modifier) and right click on group tab will remove group.</li>
<li>Support for QT5 lv2 UIs in bundles/installers from ardour.org</li>
<li>Add support for relocating missing external files in Missing File dialog.</li>
<li>Improvements to wording in Preferences and Session Properties.</li>
<li>Change master meter type in toolbar via right click.</li>
<li>Constrain VCA slave value to prevent overshoot and snap back on release.</li>
<li>Use consistent Installer names for all platforms (Name + Version + Arch)</li>
  </ul>

  <h2>MIDI</h2>
  <ul>
<li>Separate bank selection into MSB/LSB in Patch Change Dialog.</li>
  </ul>

  <h2>Plugins</h2>
  <ul>
<li>Allow rescanning of VST plugins after changing VST path.</li>
  </ul>

  <h2>Scripting</h2>
  <ul>
<li>Update bindings and scripts for API changes.</li>
<li>Updates to Lua scripting documentation.</li>
<li>Add bindings for plugin reordering and add example reverse plugin order script.</li>
<li>Add bindings for solo/mute and realtime control changes and update example script.</li>
<li>Bind scripts via right click on Lua Action Script Button in Toolbar.</li>
<li>Unbind scripts via Shift + right click on Lua Action Script Button in Toolbar.</li>
<li>Allow scripts to define a custom icon when bound to a Lua Action Script Button.</li>
<li>Include Audio to MIDI script in Bundles/Installers.</li>
<li>Add ability to Load/Remove Session scripts in the Script Manager.</li>
<li>Edit button in Script Manager now opens Lua Script Window.</li>
<li>Add bindings to create Aux sends and add script to send multiple tracks to an Aux bus.</li>
<li>Add Action Script Button icons for various scripts.</li>
<li>Add example script to show/hide Tracks in the Editor.</li>
  </ul>

  <h2>Translation Updates</h2>
  <ul>
<li>Updated Japanese translation by Hiroki Inagaki</li>
<li>Updated Czech translation by Pavel Fric</li>
  </ul>

  <h2>Fixes</h2>
  <ul>
<li>Use correct position when copying MIDI regions</li>
<li>When moving MIDI regions with non-default meter and snap set to Beats, finish in the corect position [#7234]</li>
<li>Reset window size when switching between custom and generic plugin UI. </li>
<li>Fix timing of regions recorded before the musical origin (bar 1, beat 1)</li> loaded correctly.</li>
<li>Vertically constrained Region drag now working again.</li>
<li>Properly update main window title when switching Snapshots.</li>
<li>Toolbar height no longer changes with varying appearance options.</li>
<li>Fix issues with Track and Mixer Strip order.</li>
<li>Don't show hidden Mixer Strips when spilling a VCA.</li>
<li>Undo/Redo now works for Snap position to grid via Region context menu. [#7246]</li>
<li>Ensure toggle Editor & Mixer action always works with a detached Mixer Window.</li>
<li>Percussive notes now shown in correct position while dragging during move/copy.</li>
  </ul>
