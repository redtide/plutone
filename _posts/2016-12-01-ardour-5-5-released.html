@paul wrote:

  <p>
Ardour 5.5 is now available, with a variety of new features
and many notable and not-so-notable fixes. Among the notable
new features are support for VST 2.4 plugins on OS X, the
ability to have MIDI input follow MIDI track selection,
support for Steinberg CC121, Avid Artist &amp; Artist Mix
Control surfaces, "fanning out" of instrument outputs to new
tracks/busses and the often requested ability to do horizontal
zoom via vertical dragging on the rulers. There are also the
usual always-ongoing improvements to scripting and OSC
support.
  </p>
  <p>
As in the past, some features including OSX VST support,
Instrument Fanout, and Avid Artist support were made possible
by sponsorship from Harrison Consoles.
  </p>

  <h3 class="section-header">Fixes</h3>
<ul>
  <li>Work around MIDI devices sending invalid timestamps on OSX.</li>
  <li>Open Sessions with files referenced from removable drives on Windows.</li>
  <li>Various fixes for NetBSD and FreeBSD.</li>
  <li>Add MIDI Control Change at correct location when importing.</li>
  <li>Update Region colors on theme change.</li>
  <li>Fix/Support note name translations (e.g "Ré ♯").</li>
  <li>Allow loading sessions with missing external files. [#7067, #7114]</li>
  <li>Use distinct color identifier for meter clip indicator. [#7010] </li>
  <li>Prevent panners being reset when switching to Aux sends and back. [#6893]</li>
  <li>Use correct BBT offset when using Insert Time. [#7072]</li>
  <li>Fix restoration of key bindings assigned to Windows+key. [#7037]</li>
  <li>Prevent duplicate moves when selecting regions on shared playlists. </li>
  <li>Fix crash when instrument instantiation fails on track creation. </li>
  <li>Prevent infinite loop when calculating tick marks on low zoom levels.</li>
  <li>Fix LTC-generator 24h wraparound</li>
  <li>Correctly handle failed activation of Control Surface Modules</li>
  <li>Fix Master Track moving down one position in Editor after Session reload. [#7080]</li>
  <li>Mark Session state as changed when tracks are reordered. [#7093]</li>
  <li>Improved fix for playback of linked MIDI Regions [#6541]</li>
  <li>Fix to keep Meterbridge Window on top.</li>
  <li>Fix for non-working buttons in Monitor Section after second Session load [#7098]</li>
  <li>Fix LV2 state versioning with Session templates.</li>
  <li>Include Aux Busses in default strip types in OSC Surface [#7090]</li>
  <li>Fixes to Insert/Remove time dialog [#7072]</li>
  <li>Fix intermittent hang when stopping PortaudioBackend [#7095] </li>
  <li>Fix playlist XML nodes changing order on Session save [#7053]</li>
  <li>Fix crash loading u-he Diva VST plugin on Windows</li>
  <li>Fix Duplicating Tracks with Sidechains [#7117]</li>
  <li>Fix incorrect Midi Region length when splitting</li>
  <li>Fix for widget visibility with detached preferences window [#7002]</li>
  <li>Fixes when creating MIDI notes in Sustained note mode [#7126]</li>
  <li>Fixes when creating MIDI notes in Percussive note mode [#7130]</li>
  <li>Fixes to Cut Mode to make the cut line visible and respect snap modifier</li>
  <li>Fix inaudible Percussive notes being displayed at the end of the Region</li>
  <li>Fix crash when editing pitch of notes to outside visible area [#7135]</li>
  <li>Fix crash when deleting a MIDI Region while mouse over CC event</li>
  <li>Fix concurrency/locking issue with Event Loop request invalidation [#7049]</li>
  <li>Fix creation of playlists with duplicate names [#6523, #5681, #6497]</li>
  <li>Fix output display for non-plugin processors in the Processor Box</li>
  <li>Fix forwarding of output buffers by using correct channel count</li>
  <li>Fix loading, recording & saving MIDI with PolyKeyPressure events [#6948]</li>
  <li>Fixes to Duplicate Track [#7141, #7146]</li>
  <li>Fixes to OSC Control Surface</li>
  <li>Fix handling of pitch bend events by a-fluidsynth [#7140]</li>
  <li>Fix for reading CAF audio files over a certain size</li>
</ul>

  <h3 class="section-header">General Improvements</h3>
  <ul>
<li>Many MIDI and Tempo related refactorings, fixes and improvements to performance, source code documentation and tests. (Thanks Nick)</li>
<li>Include Session utilities in the application installers/bundles.</li>
<li>Allow feedback loops from internal Sends to enable custom "Echo" chains.</li>
<li>Improved performance when reading and writing XML files.</li>
<li>Improved performance of Region export.</li>
<li>Snap to Track Strip boundaries when scrolling with arrow keys in the Editor.</li>
<li>Clicking on a Strip in the Mixer Strip List will move the Strip into view.</li>
<li>Snap to Mixer Strip boundaries when scrolling with arrow keys in the Mixer</li>
<li>Minor updates to Cubasish Theme.</li>
<li>Compensate for latency/correct alignment when exporting the Master Bus.</li>
<li>Improve automatic Track naming [#7092]</li>
<li>Add option to zoom in the time rulers using button press and dragging vertically. [#6768]</li>
<li>Speed up changes to AFL/PFL on large Sessions</li>
<li>Large improvements in time to process Solo changes [#7119] </li>
<li>Speed up Track/Bus creation time</li>
<li>Restrict ALSA input/output device selection to valid combinations</li>
<li>Speed up Track deletion when Editor Mixer Strip is visible</li>
</ul>

  <h3 class="section-header">MIDI</h3>
  <ul>
<li>Many minor fixes and improvements</li>
<li>Support MIDI input follows Track selection. [#6744]</li>
<li>New MIDI -> Ports page in Preferences for managing MIDI port functionality and enabling MIDI input follows Track Selection.</li>
<li>Added support for Type 0 MIDI files and add option to split channels into tracks on import.</li>
<li>Added MIDI mapping for AKAI Midimix.</li>
<li>New MIDI Note selection XML format, existing note selections will be lost.</li>
<li>Improved editing in Percussive note mode</li>
</ul>

  <h3 class="section-header">Plugins</h3>
  <ul>
<li>VST 2.4 plugin support on OSX</li>
<li>Add functionality to Fan out an Instruments outputs to new Tracks/Busses (video demo explanation at <a href="https://vimeo.com/191024580">https://vimeo.com/191024580</a>)</li>
<li>Allow LV2 plugins to provide Midnam XML.</li>
<li>Increase maximum release time in a-comp.</li>
<li>Sort Instrument list alphabetically.</li>
<li>Updated internal versions of qm-dsp library and qm-vamp plugins.</li>
<li>Allow LV2 Plugins to signal state changes and mark Session state as changed.</li>
<li>Support LV2 PortGroup extension</li>
<li>Add support for output-channel/bus grouping in AU plugins</li>
</ul>

  <h3 class="section-header">OS X</h3>
  <ul>
<li>Mouse drags on plugin GUIs (editors) are now handled
  correctly when they end outside the window.</li>
<li>Video monitor "full screen" mode no long spans all screens</li>
  </ul>

  <h3 class="section-header">Scripting</h3>
  <ul>
  <li>Added or improved Lua bindings for VAMP plugins, TempoMap, Basic MIDI model editing, Inter-Processor communication, Source, AudioSource, Route, Panner, Instrument</li>
  <li>Add example script for Audio to Midi Transcription.</li>
  <li>"Sound Smasher" example script added.</li>
  <li>Add prototype script for converting polyphonic audio region to a MIDI region.</li>
  <li>Add example script for instrument fan-out</li>
  <li>Add example script to modulate parameters globally</li>
  <li>Add support for calling ARDOUR.DSP Lua functions from UI scripts</li>
  <li>Add example script to process Regions selected in the Editor</li>
  <li>Add MIDI LFO example script</li>
  </ul>

  <h3 class="section-header">Control Surfaces</h3>
  <ul>
<li>Added Support for Steinbergs CC121 Control Surface</li>
<li>Activate/Deactivate plugins via OSC</li>
<li>Added configuration files for the Avid Artist Control and Artist Mix Control surfaces.</li>
<li>Allow Aux Buses visibility to be controlled separately via OSC</li>
<li>Add OSC support to query strips internal sends [#7136]</li>
<li>Add OSC support to query plugins and their parameters [#7137]</li>
  </ul>

  <h3 class="section-header">Translations</h3>
  <ul>
<li>Updated German translation (Edgar Aichinger)</li>
<li>Updated French translation (Olivier Humbert)</li>
</ul>
